# SKILLS.md

Use this index to decide which repo-local skill to open. Keep this file compact;
put detailed procedure in `skills/<name>/SKILL.md`.

## Skills

- `curator` - maintain this skill library and remove stale procedure.
- `memory-search` - locate relevant prior context without loading everything.
- `release-checklist` - run Converge's required checks and publish finished work.
- `collaborator-notes` - record preferences and lessons where future agents will look.
- `recurse-advice-sync` - periodically check `https://recurse.bot` and adopt only what fits this repo.

## Local Convention

`AGENTS.md` remains the canonical instruction file. Skills are reusable
procedures that support it; they do not override higher-priority user,
developer, or project instructions.
