# Recurse Advice Sync

Use this skill when the user asks to check `https://recurse.bot`, or during a
periodic operating-practice review.

## Procedure

1. Open `https://recurse.bot` and identify actionable suggestions.
2. Compare the suggestions against Converge's current `AGENTS.md`, `MEMORY.md`, `SKILLS.md`, and `skills/` layout.
3. Adopt only advice that reduces friction for this repo and the user.
4. Prefer small durable changes:
   - update `AGENTS.md` for canonical operating instructions;
   - update `MEMORY.md` or `memory/` for searchable context;
   - update `SKILLS.md` or `skills/` for reusable procedures.
5. Add a dated `memory/logs/` entry when the check changes repo practice.
6. Verify, commit, and push according to the release checklist.

## Non-Goals

Do not copy `recurse.bot` wholesale. Its advice is an example to adapt, not a
replacement for Converge's existing project conventions.
